English Dictionary project

Build and Run
1. Build the project
   mvn clean package
2. Run the program
   java -jar dict-1.0.jar

Scenario
1. Select a random word from DB
2. Show his translation to an user
3. Show five random words to user
4. User should to input a original word
5. User interface like http://actvoc.narod.ru/
